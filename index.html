THIS SITE IS FOR DEMONSTRATION PURPOSES ONLY!!!

The site is based on the data8.org syllabus from Spring 2020. The goal is to illustrate Jupyter Notebook pages in these providers: a demonstration JupyterHub instance, Google Colab and Deepnote. These Notebook pages use Otter Grader instead of OKpy to auotmatically grade and provide feedback to the student. Documentation for Google Colab is here; Deepnote is here. The source files used in JupyterHub and Google Colab are in these Github repositories: source files for Jupyterhub and source files for Colab.
